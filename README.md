===============================
📘 STUDENT RECORD SYSTEM (HTML)
===============================

📂 Project Type:
---------------
Frontend-only Web App (HTML, CSS, JavaScript)

🌐 Live in Browser:
-------------------
This project runs 100% in your browser. No backend or server required.
Open `index.html` in Chrome or any modern browser.

🎯 Features:
------------
✔ Add student records (ID, Name, 5 subject marks)
✔ Automatic GPA calculation
✔ Grade assignment (A+, A, B, C, F)
✔ Responsive table with data display
✔ Sort by GPA or Name
✔ Data stored using LocalStorage (persists after refresh)
✔ Dark Mode toggle
✔ GPA Bar Chart (Chart.js)
✔ Grade Pie Chart (Chart.js)
✔ Mobile responsive design

🔧 Technologies Used:
---------------------
- HTML5
- CSS3 (inline styles)
- JavaScript (Vanilla JS)
- Chart.js (via CDN)
- LocalStorage API

📦 How to Run:
--------------
1. Open VS Code
2. Paste the code into a file named `index.html`
3. Right-click the file → Open with Live Server (or open manually in browser)
4. Start adding student data!

📈 Chart Visuals:
------------------
- GPA Chart: Visual bar chart of all student GPAs
- Grade Chart: Pie chart showing grade distribution (A+, A, B, etc.)

📁 Files:
---------
- `index.html` → Full application code (self-contained)
